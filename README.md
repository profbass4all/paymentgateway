## Getting Started

- **Clone the Repository**:
  
  ```
  git clone < repository- url >
  ```

- **Install the Dependencies**:

  ```
  npm install
  ```

- **Run the Application**

  ```
  npm run dev
  ```

- **Access the Api**: Use tools like Postman or Thunderclient to interact with the endpoints
- **Check complete jsdoc at /api-docs**
